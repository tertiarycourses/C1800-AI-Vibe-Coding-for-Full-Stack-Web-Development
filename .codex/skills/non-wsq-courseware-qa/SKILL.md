# Non-WSQ Courseware QA

1. Confirm PPT, Learner Guide DOCX/Markdown, Lesson Plan and one lab per planned activity exist.
2. Confirm course title, code, topic order and total instructional duration agree across artifacts.
3. Confirm every lab has a goal, numbered procedure, test checkpoint and evidence requirement.
4. Search learner deliverables for SSG, TRAQOM, SkillsFuture funding and formal assessment programme content; report file and line.
5. Confirm every custom skill, command, hook and agent filename begins `non-wsq-` and no existing WSQ artifact changed.
6. Open and render PPT/DOCX files; inspect for overflow, clipping, blank pages and unreadable contrast.
7. Report pass/fail with exact artifact counts and actionable locations.
