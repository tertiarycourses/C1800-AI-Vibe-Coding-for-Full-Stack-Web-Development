# Non-WSQ Courseware Build

1. Read the published course page and preserve its title, code, duration, level and topic sequence.
2. Create at least 20 connected labs with Goal, Prerequisites, numbered Steps, Verification, troubleshooting, boundary challenge, Evidence and Reflection.
3. Use the adult-learning rhythm: trainer demonstration, guided practice, independent challenge, peer explanation and reflective checkpoint. Apply the Agentic AI Loop: Frame, Plan, Generate, Inspect, Verify, Correct, Commit.
4. Build a 16:9 white-theme PPT of more than 100 slides and an Arial DOCX Learner Guide detailed enough to execute without trainer improvisation. Include Tertiary branding, version control, TOC and page footers.
5. Exclude WSQ funding, SSG, TRAQOM and assessment sections.
6. Prefix every reusable skill, command, hook and agent with `non-wsq-`.
7. Keep project tooling in project-scoped folders and user tooling in user-scoped folders; never overwrite an unprefixed or `wsq-*` artifact.
8. Run an iterative build, render, inspect and revise loop before delivery.
